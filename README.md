# Coastal Creations Studio

## About

Coastal Creations Studio is a vibrant art studio owned by Ashley Mathers, offering a variety of creative classes, workshops, camps, and birthday parties for all ages. The platform also runs a full online store for physical products (Art Kits and more) — a Square-catalog-driven storefront with live Shippo shipping rates, automated label purchase, and carrier-tracking-driven order updates.

## 💻 Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) - React 19 framework for production
- **Database:** [MongoDB](https://www.mongodb.com/) with Mongoose ODM
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) & Material UI
- **Authentication:** NextAuth with Google OAuth
- **Payments:** [Square](https://squareup.com/) Web Payments SDK + Catalog API (store products)
- **Shipping:** [Shippo](https://goshippo.com/) for rates, labels, and tracking webhooks
- **CMS:** [Sanity](https://www.sanity.io/) for content management
- **Email:** [Resend](https://resend.com/) API for transactional emails
- **Hosting:** [Vercel](https://vercel.com/)
- **Language:** TypeScript
